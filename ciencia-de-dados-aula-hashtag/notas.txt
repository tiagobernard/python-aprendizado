Pré-processamento
A inteligencia artificial sempre aprende com números.
Sempre substituir palavras em números

Treino é o que a IA vai usar para aprender e teste o nome já diz, pra testar
Maioria dos dados são usados para aprender.

Criar um modelo de IA
O melhor modelo a gente não sabe qual é, até testar o modelos.
Mas sabemos como chegar na lista de modelos que devemos testar
São testados os modelos de acordo com nossos desafios.
Modelo de classificação, tipos de dados do lado A e dados do lado B. Separadno em varias categorias
Modelo de Regressão, é o modelo para quanto é a passagem aérea, ex. previsão

Aprendizado da IA
Supervisionado, é o que vc tem a resposta com base em um histórico.
Não Supervisionado, é quando vc não tem resposta certa, ex. separar o perfil de cliente.
Reforço, é quando a IA começa burra, quando faz a começa a cosia certa é recompensado. Quanto mais a IA faz certo, ela ganha um ponto. Ex. recomendação de vídeo do YT.

Escolha do melhor modelo
Regressão Linear(e variações)
Regressão com rede neural
Árvore de decisao
Randomforest
Regressao KNN
SVM
Treinar cada uma desses modelos.
A escolha é feita para o modelo que erra menos.
